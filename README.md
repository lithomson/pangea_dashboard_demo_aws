# pangea_dashboard_demo_aws
![Python 3.6 to 3.8](https://github.com/lithomson/pangea_dashboard_demo_aws/workflows/Python%203.6%20to%203.8/badge.svg)

## Requirements

- Python 3
- Docker
- Dash
- AWS account
- AWS credentials configured

## Build the app

`sam build --use-container`

## Deploy the app

`sam deploy`
